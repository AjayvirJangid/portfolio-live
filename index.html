Enemy AI

I developed an Enemy AI system in Unreal Engine 5 using Blueprints, focusing on behavior logic and combat interactions. The AI is capable of patrolling, detecting the player, and engaging in combat, with dynamic health bars to visually track status. It responds to player actions, takes damage, and can be defeated, transitioning into a death state as shown in the scene. The system also utilizes navigation meshes for pathfinding and includes state-based logic for idle, chase, and attack behaviors. This project highlights my ability to design intelligent gameplay systems and implement responsive AI mechanics within Unreal Engine.

  • Enemy AI / Unreal Engine

Multiplayer Game

I recently completed a cross-platform multiplayer game project in Unreal Engine 5, gaining hands-on experience with both Blueprints and C++. The project includes core gameplay systems such as player movement, shooting, interaction mechanics, health and damage synchronization, and a functional HUD with health bars, crosshairs, and notifications. On the networking side, I implemented Unreal’s replication system, online subsystem setup, and a complete lobby system with session management, tested on both PC and Android. I also designed polished UI/UX using UMG widgets, menus, and animations while resolving issues like duplicate UI loading. For deployment, I successfully packaged the game for Android with proper APK and OBB handling, resolving manifest and permission errors. This project strengthened my skills in UE5, multiplayer replication, UI systems, and mobile deployment, providing me with valuable end-to-end game development experience.
